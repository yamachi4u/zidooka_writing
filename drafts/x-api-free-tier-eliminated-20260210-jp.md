# X APIが事実上の無料枠廃止へ - 従量課金制に正式移行

X（旧Twitter）が2026年2月6日、開発者向けAPIの新しい従量課金モデル（Pay-Per-Use）を正式に発表しました。これまでの月額200ドルまたは5000ドルの固定料金制から移行し、事前に購入したクレジットがAPIリクエストに応じて消費される方式になります。

:::conclusion
**X APIの無料枠は事実上廃止されました。** 2023年2月の無料提供終了に続き、2026年2月に従量課金制が正式導入。個人開発者や小規模プロジェクトにとって、API利用のハードルが大幅に上がっています。
:::

## 従量課金制の主な変更点

### 料金体系の変更

これまでの月額固定制（Basic: $200/月、Pro: $5,000/月）に代わり、**事前にクレジットを購入して使う従量課金制**が導入されました。APIエンドポイントごとに単価が異なります：

| 操作 | 料金 |
|------|------|
| 投稿の読み取り | $0.005/件 |
| ユーザー情報取得 | $0.010/件 |
| 投稿の作成 | $0.010/件 |

X社の試算例では、適度な利用量で月約$215（約3.3万円）程度かかる見込みです。

:::warning
同一の投稿やユーザーを同日に繰り返し取得する場合、原則として重複分の課金は回避される仕組みがありますが、**完全な保証ではない**とXは明言しています。
:::

### 無料で使える例外

:::note
以下の場合のみ、無料でスケールされたアクセスが可能です：

- **公益性が高い「公共アプリ（Public Utility apps）」**
- 旧無料枠を利用していた開発者には**1回限り$10（約1,570円）のバウチャー**が付与
:::

## 経緯と背景

### 2023年2月：無料提供終了

XはTwitter時代の2023年2月、「Twitter API v2」および「Twitter API v1.1」の無料提供を終了し、有料プランに一本化すると発表しました。

### 2025年10月：従量課金制のベータ提供

2025年10月頃から、Pay-Per-Useモデルをクローズドベータで提供していました。これが2026年2月に正式リリースされました。

### 固定料金プランの継続

既存のBasic（$200/月）・Pro（$5,000/月）プランは、すでに契約している開発者向けに継続可能です。ただし、新規開発者は原則として従量課金制がデフォルトとなります。

## 開発者への影響

:::warning
個人開発者や学生、小規模スタートアップにとって、API利用が事実上困難になりました。Botや研究ツール、サードパーティーアプリの開発・運用が大きく制限されています。
:::

### 利用管理機能

使いすぎを防ぐため、以下の設定が可能です：

- **自動クレジット追加**：残高が一定以下になったら自動購入
- **支出上限設定**：請求サイクルごとの上限を設定し、到達時にリクエスト停止
