API版とWEB版について、実運用で重要になるポイントを比較表で整理します。

:::conclusion
納入前提なら、規約グレー運用を避けられるAPI版か、明確にウォーターマークなしを許可する上位プランを選ぶのが安全です。
:::

## 比較表

| 区分 | API版 | WEB版 Pro | WEB版 Ultra |
| --- | --- | --- | --- |
| 料金体系 | 従量課金 | サブスク | サブスク |
| 目安コスト | 単価40円（失敗含むと約100円になる可能性） | 3,000円 | 36,000円 |
| 4K出力 | 可能 | 可能 | 可能 |
| ウォーターマーク | なし | あり（常時） | なし |
| 納入観点 | ルール設計次第で運用しやすい | 後消し前提は規約違反リスクがありグレー | クリアに運用しやすい |

## 判断の目安

:::note
小ロットやテスト中心なら、API版の従量課金は初期コストを抑えやすいです。
:::

:::warning
WEB版Proでウォーターマークを後から消す運用は、規約違反に該当する可能性があるため、納入用途では避けるのが無難です。
:::

:::step
納入が前提なら、契約プランの利用規約で「商用利用時のウォーターマーク扱い」を先に確認してから選定してください。
:::
