最近あらためて感じているのは、AIで人間の仕事がなくなった、というより、==仕事の重心がかなり変わった==ということでした。

前は自分がやっていたことの多くが、作業そのものでした。

情報を集める。
画面を見に行く。
並べる。
叩き台を書く。
同じ説明を少しずつ変えながら整える。

もちろんその全部に意味はあります。
ただ、そこで使っていた時間のかなりの部分は、「自分しかできない判断」ではありませんでした。

今は API やエージェントを組み合わせることで、その前段の作業をかなり任せられるようになってきました。

:::conclusion
減ったのは仕事そのものではなく、手作業の比率で、代わりに人間側の役割は「整える人」「決める人」に寄ってきました。
:::

この変化は、思ったより大きいです。

たとえば資料づくりでも、以前は「材料を集める人」と「文章を起こす人」を自分ひとりで行き来していました。
今は、材料の取得や初稿づくりはかなり前に進めてもらえて、自分はそのあとで、

1. 何を主張の軸にするか
2. どこを省くか
3. 相手にどう見せるか
4. 本当にその解釈でいいか

を見ればよくなっています。

ここは結局、人間がいちばん責任を持つべき部分でもあります。

AIが強いのは、早く出すこと、広く拾うこと、叩き台を作ることです。
一方で、何を残して何を切るか、どこを強く言うか、相手に合わせて温度感を変えるか、という編集判断はまだかなり人間側に残っています。

だから体感としては、「仕事が奪われる」というより、「雑に言えば下ごしらえが外に出た」感覚に近いです。

この状態になると、求められる能力も少し変わります。

以前より大事になるのは、

* どの論点を前に出すか
* どういう順番で見せるか
* 事実と解釈をどこで切り分けるか
* 最後の一文をどう締めるか

のような編集の力でした。

言い換えると、手を動かす総量が減っても、判断の密度はむしろ上がります。

ここが面白いところで、AIを入れたのに仕事が軽くなりすぎる感じはあまりありません。
むしろ「何を出すべきか」を問われる場面が増えるので、別の意味で仕事の質が上がっている気がします。

たぶん今起きているのは、自動化というより再配置です。

人間は前工程の単純作業から少しずつ離れて、編集、判断、責任の側に寄っていく。
この変化をうまく受け取れた人ほど、AIと仕事するのが自然になるのだと思います。
