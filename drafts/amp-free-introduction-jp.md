AIコーディングエージェント「Amp (Ampcode)」が、とんでもない太っ腹なプランを発表しました。その名も **"Amp Free"** です。

## 毎日10ドル（約1,500円）分のクレジットが無料

Amp は VS Code などで動くAIエージェントですが、新しい無料プランの衝撃度は群を抜いています。

- 【結論】**毎日10ドル（月額換算で約300ドル/45,000円相当）** のクレジットが無料で付与されます。
- 【ポイント】このクレジットを使えば、**Claude Opus 4.5** などの最先端モデル（Frontier Models）をフル機能で利用可能です。

## 使えるモデルが豪華すぎる

2026年1月現在のドキュメントによると、以下のモデルが統合されています。

1.  **Smart Mode:** **Claude Opus 4.5** で駆動（メインのコーディング）
2.  **Rush Mode:** **Haiku 4.5** （高速処理用）
3.  **Oracle:** **GPT-5** （推論・プランニング用）
4.  **Subagents:** **Gemini 3** （サブタスク実行用）

これらを適材適所で使い分けるエージェントワークフローを、個人開発者が無料で試せる時代が来ました。広告収入（Ad-supported）によって成り立っているとのことです。

## 注意点：本人確認が厳しい

当然ながら、毎日10ドルを配るというのはサービス側にとって莫大なコストです。そのため不正利用対策が徹底されています。

> 【注意】本人確認（Identity Verification）が非常に厳格です。
> 捨てアドや匿名性の高い登録方法では、無料枠（Grant）の付与が拒否されるケースが報告されています。

「あなたが実在する人間であること」を証明するために、本名の使用やGitHubアカウントの信頼性が重要になるようです。

## 始め方

1.  [ampcode.com](https://ampcode.com/) にアクセス。
2.  GitHubアカウント等でサインアップ。
3.  VS Code拡張機能やCLIツールをインストール。

Opus 4.5 や GPT-5 クラスのモデルを無料で使い倒せるチャンスですので、興味のある方は早めに登録しておくことをおすすめします。
